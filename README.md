3.Move to  Flutter Web Build Directory 
  ```
  git init
  git add .
  git commit -m 'first'
  git branch -M main
  git remote add origin https://github.com/johmartin/flutterwebadmin.git 
  git push -u origin main
  ```
  password : input token value, 위에서 생성한 토큰값 입력
